# Blicky Wally

A minimal dark-themed wallpaper gallery.

## Setup

1. Upload `index.html` to your repository
2. Go to your repository Settings → Pages
3. Set Source to "Deploy from a branch"
4. Select "main" branch and "/ (root)" folder
5. Save and wait a few minutes for deployment

Your site will be live at: `https://666forever.github.io/blicky_wally/`

## Features

- Clean, minimal dark design
- Responsive grid layout
- Mobile-friendly
- Click any image to open full size
- Download button on hover (always visible on mobile)
- 300+ aesthetic wallpapers

## Usage

- Click any wallpaper to view full size in new tab
- Hover over (or tap on mobile) and click the download button (↓) to save
